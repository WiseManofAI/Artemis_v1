# Artemis_v1


Offline presence-based cognitive agent.

## Features
- No preset dialogue
- Organic language generation
- Multi-action planning
- Long-horizon presence forecasting
- Experience-based learning
- Emotion-modulated reasoning
- Persistent memory
- Hard failsafe

## Commands
- Die       → terminate
- Die:Re    → reset memory
- Stats    → view internal state

## Run
```bash
pip install -r requirements.txt
python artemis.py
